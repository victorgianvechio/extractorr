Introduction

A web scrap application developed with MEAN Stack and IBM Watson Language Understanding API services.

Xtractorr é uma ferramenta para analisar, extrair e descobrir entidades e conceitos de informações extraídas de sites. Este protótipo foi desenvolvido inicialmente para atuar na página do Inova Marília. A ferramenta possuí três principais funcionalidades:

Scan

O Scan é responsável por analisar a página alvo e iterar entre as páginas subsequentes extraíndo o título e o link da publicação.

Scrap

O Scrap itera os link obtidos pelo scan, extraíndo informações página por página. As informações a serem extraídas são mapeadas na ferramente, tais como: conteúdo da publicação, data da publicação, categoria, etc.

Understand

Aqui é feito as chamadas para a API do IBM Watson Language Understanding, onde é enviado o texto/conteúdo da publicação para que o Watson descubra entidades, conceitos e atribua relevância para os mesmos.

Ao final temos um arquivo JSON contendo todo o conteúdo extraído e as informações processadas pelo Watson. O código fonte está disponível no github.
